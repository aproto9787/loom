Status

Experimental, local-first, and code-backed.

Heddle currently runs recursive YAML-defined agent trees with local Claude Code and Codex processes, MCP delegation, Studio, and persisted traces.

Live

Implemented

  • Recursive agent-tree YAML schema.
  • Host and isolated runtime metadata.
  • Run-scoped Heddle MCP delegation tools.
  • Studio flow editing and run observation.
  • SQLite event and report persistence.
  • Published npm package at @aproto9787/heddle.

Safety

Run it only in trusted workspaces.

Heddle can launch local CLI tools, provider sessions, and shell hooks. Current local execution paths can use powerful approval and sandbox bypass flags. Treat flows, hooks, roles, skills, and MCP configuration as trusted code.

Source of truth

Detailed state stays in the repository.

Read the code-backed checklist in the repository docs directory before documenting new guarantees.
